TallySync is a Windows desktop app built with Python (Tkinter) that automatically syncs financial data from Tally ERP into a MySQL database.
It connects to Tally's XML API, pulls vouchers, ledgers, and trial balance data, and stores them in structured DB tables — supporting both full snapshots and incremental CDC syncs via Tally's alter_id.
The app runs a background scheduler (APScheduler) that fires sync jobs on a per-company schedule (every N minutes, hours, or daily). Multiple companies are processed one at a time through a FIFO queue — Tally opens automatically for each company via UI automation, syncs, then moves to the next. If the app is closed and reopened, missed syncs are detected and caught up automatically on startup. The app minimizes to the system tray so the scheduler keeps running even when the window is closed.
